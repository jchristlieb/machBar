## Über das Projekt

Die **machBar** ist eine genossenschaftlich organisierte Kneipe in Wittenberge (Brandenburg). Sie wird als einzige rauchfreie Kneipe der Stadt ein offenes Angebot für Einheimische, Touristen und Durchreisende schaffen — getragen von einer Genossenschaft (eG) mit aktuell 37 Mitgründer:innen.

- **Gründungstermin:** 8. Juni 2026
- **Eröffnung:** 1. September 2026
- **Website:** <https://machbar-wittenberge.de>
